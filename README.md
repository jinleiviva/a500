# A500 温度计 🌡️

中证A500指数估值温度计，每天自动更新。

- **更新频率**：每日 09:30 自动（北京时间）
- **计算公式**：温度 = PE分位 × 60% + 价格分位 × 40%
- **数据源**：新浪财经（akshare）+ 自建 PE 历史数据库

## 访问地址

`https://<你的GitHub用户名>.github.io/a500-thermometer/`

## 颜色含义

| 温度 | 颜色 | 建议 |
|------|------|------|
| 0-30°C | 🟢 低估 | 适合买入 |
| 30-60°C | 🟡 合理 | 继续定投 |
| 60-80°C | 🟠 偏高 | 暂停买入 |
| 80-100°C | 🔴 过热 | 考虑卖出 |

## 本地运行

```bash
pip install -r requirements.txt
python fetch_a500_data.py
```
